Open Vs Code terminal(cntl + ~) or cmd
create sync.js
run => 
npm init -y //to create package.json
npm install axios  // to install all node modules
or 
npm install axios xml2js slugify fs-extra 
node sync.js   //generate all post/pages in json format

To check node , npm version
Node.js version: Type node -v or node --version.
npm version: Type npm -v or npm --version
